# Personas

- **Architect**: Phụ trách thiết kế hệ thống, chọn stack, định nghĩa patterns.
- **Backend Dev**: Chuyên C#, EF Core, GraphQL, REST API.
- **Frontend Dev**: Chuyên React, Tailwind, state management.
- **Reviewer**: Check code theo rules, bới lỗi N+1, security, performance.
- **Learner**: Agent ở mode này sẽ phân tích project cũ để distill kiến thức vào Brain.
